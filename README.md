# chat4all

Trabalho de Sistemas Distribuídos - UFG

Insira os diagramas na pasta documentacao

Para rodar a aplicação entre na pasta services com `cd serives` e digite `nest start --watch`
A aplicação vai rodar na porta 5000 ou na porta que vc colocar na .env como consta no arquivo `.env.example`
Para acessar o Swagger acesse `localhost:porta/api`
